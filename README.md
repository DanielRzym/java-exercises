## üöÄ How to Run
To run any of the files in this repository, follow these steps in your terminal:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/DanielRzym/java-excersises.git](https://github.com/DanielRzym/java-excersises.git)
    cd java-excersises
    ```

2.  **Compile the specific file:**
    ```bash
    javac NumberUtils.java
    # Or for future files: javac FileName.java
    ```

3.  **Run the application:**
    ```bash
    java NumberUtils
    # Or for future files: java FileName
    ```
